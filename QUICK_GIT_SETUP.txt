QUICK GIT & GITHUB SETUP INSTRUCTIONS
================================================================================

Your repository is ready! All files are staged. Follow these steps:

STEP 1: Configure Git (Replace with YOUR information)
------------------------------------------------------
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

STEP 2: Create Your First Commit
----------------------------------
git commit -m "Initial commit: Discord clone with auto-friend feature"

STEP 3: Create GitHub Repository (if you don't have one)
---------------------------------------------------------
1. Go to: https://github.com/new
2. Repository name: Dosco (or any name you want)
3. Make it Public or Private (your choice)
4. DO NOT check "Initialize with README"
5. Click "Create repository"

STEP 4: Add GitHub Remote
--------------------------
Replace YOUR_USERNAME and YOUR_REPO_NAME with your actual values:

git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

Example:
git remote add origin https://github.com/johndoe/dosco.git

STEP 5: Push to GitHub
-----------------------
git push -u origin main

If you get authentication error, you'll need a Personal Access Token:
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Check "repo" permission
4. Copy the token
5. When pushing, use token as password (username = your GitHub username)
